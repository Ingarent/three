    • Introduction

    Python is a cross-platform, high-level interpreted programming language. It is known for its simplicity and readability, making it an ideal choice for both beginners and experienced developers. Python can be used in various environments, such as web development, data analysis, artificial intelligence, automation, and more.

    Python includes a standard library of objects, such as lists (list), dictionaries (dict), dates (datetime), and mathematical functions (math), as well as core language elements like operators, control structures, and statements. The core of Python can be extended for various purposes using additional modules and libraries. For example:

    • Web Development: With frameworks like Django and Flask, Python enables the creation of powerful web applications and APIs.
    • Scientific Computing: Libraries such as NumPy, SciPy, and Pandas provide tools for data manipulation and complex computations.
    • Automation: Python can be used to write scripts that automate repetitive tasks.
    What You Should Know

    This guide assumes you have the following basic background:

    • A general understanding of programming fundamentals.
    • Familiarity with basic concepts such as variables, loops, and functions.
    • If you are new to programming, it is recommended to start with basic tutorials.
    Python and Other Languages

    Python is often compared to languages like JavaScript and Java. While Python shares some syntactic similarities with these languages, it is fundamentally different in its approach. Python is dynamically typed, meaning you do not need to declare variable types explicitly. It also uses an object-oriented paradigm but is more flexible and less verbose compared to Java.

    Python’s design philosophy emphasizes code readability and simplicity, which is reflected in its use of indentation for block structuring instead of braces. This makes Python code more intuitive and easier to maintain.

    Hello, World!

    To get started with Python, you can write your first "Hello, World!" program:

    def greet_me(your_name): print(f"Hello, {your_name}!") greet_me("World")

    Simply run this code in a Python environment, and you’ll see the output in your console.

    Variables

    Variables in Python are used as symbolic names for values in your program. Variable names, called identifiers, follow certain rules:\

    • A Python identifier must start with a letter (a-z, A-Z) or an underscore (_).
    • Subsequent characters can be letters, digits (0-9), or underscores.
    • Python is case-sensitive, so myVar and myvar are considered different variables.

    Examples of valid variable names include number_hits, temp99, and _name.

    Declaring Variables

    In Python, variables are created when you assign a value to them. Unlike JavaScript, Python does not require keywords like var, let, or const to declare variables. Instead, you simply assign a value to a variable name. For example:

    x = 42

    This creates a variable x and assigns it the value 42. In Python, variables are always scoped to the block in which they are defined, such as a function, loop, or conditional statement.

    Variable Scope

    In Python, the scope of a variable determines where it can be accessed:

    • Global Variables: If a variable is declared outside of any function or block, it is considered a global variable. Global variables can be accessed from anywhere in the code, including inside functions.
      • x = 42 # Global variable def my_function(): print(x) # Accessing the global variable my_function() # Output: 42
    • Local Variables: If a variable is declared inside a function or block, it is considered a local variable. Local variables can only be accessed within the function or block where they are defined.
      • def my_function(): y = 13 # Local variable print(y) my_function() # Output: 13 print(y) # This will raise a NameError because y is not defined outside the function
    Block Scope

    Python does not have block-level scope like JavaScript. Variables declared inside a block (such as a loop or conditional statement) are accessible outside that block, as long as they are in the same function or global scope. For example:

    if True: z = 10 # This variable is accessible outside the if block print(z) # Output: 10

    However, if you declare a variable inside a function, it will not be accessible outside that function, regardless of whether it was declared in a block or not.
